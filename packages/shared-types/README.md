# @repo/shared-types

Shared TypeScript contracts for applications and packages.

## Purpose

Stores reusable DTO placeholders, utility types, and API response contracts. It intentionally contains no business entities.

## Commands

- `pnpm build`
- `pnpm lint`
- `pnpm typecheck`

## Dependencies

- TypeScript
- tsup

## Usage

Import shared contracts from `@repo/shared-types`.

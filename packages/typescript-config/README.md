# @repo/typescript-config

Shared TypeScript configuration presets for the monorepo.

## Purpose

Centralizes strict compiler settings so applications and packages use the same TypeScript baseline.

## Commands

This package has no runtime commands.

## Dependencies

No runtime dependencies.

## Usage

Extend one of the provided configs:

- `@repo/typescript-config/base.json`
- `@repo/typescript-config/nextjs.json`
- `@repo/typescript-config/nestjs.json`
- `@repo/typescript-config/react-library.json`

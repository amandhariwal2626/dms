# @repo/web

Next.js frontend foundation.

## Purpose

Provides an App Router shell, shared UI integration, TanStack Query provider, Axios client, and Zustand example store without business screens.

## Commands

- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm typecheck`

## Dependencies

- Next.js
- React
- Tailwind CSS
- Base UI through `@repo/ui`
- TanStack Query
- React Hook Form
- Zustand
- Axios

## Usage

Run from the repository root with `pnpm dev`, or from this workspace with `pnpm --filter @repo/web dev`.

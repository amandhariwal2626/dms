# @repo/api

NestJS REST API foundation.

## Purpose

Provides backend infrastructure only: application bootstrap, configuration validation, global validation, exception handling, request logging, Swagger, and health checks.

## Commands

- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm typecheck`

## Dependencies

- NestJS
- Swagger
- class-validator
- zod

## Usage

Run from the repository root with `pnpm dev`, or from this workspace with `pnpm --filter @repo/api dev`.
